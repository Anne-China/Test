## Study Notes <br>

* 文本显示格式<br>
（1）在文本下方加上=====，那上面的文本就会已大标题的格式呈现；<br>
（2）在文本下方加上-----，那上面的文本就会已中标题的格式呈现；（ - 和 + 的数量无限制）<br>
（3）当然可以有一种简单的方式来显示文本的一级标题，二级标题等格式。只要在文本前面加上对应数量的#并加一个空格即可，如一级标题一个#，二级标题##，以此类推。如下所示：<br>
# 一级标题 <br>
## 二级标题<br>
### 三级标题<br>
#### 四级标题<br>
##### 五级标题<br>
###### 六级标题<br>
